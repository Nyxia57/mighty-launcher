# ZenithMC Launcher

Launcher Minecraft moderne 

---

## Lancement

```bash
npm install
npm start
```

C'est tout. Clique "Continuer avec Microsoft" dans le launcher.

## Builder

```bash
npm run build -- --win    # Windows .exe
npm run build -- --mac    # macOS .dmg
npm run build -- --linux  # Linux .AppImage
```
